# Study Planner

A modern React + Tailwind based task management app.

## Features
- Add / Edit / Delete tasks
- Due date support
- Overdue / Today / Tomorrow status
- Smart sorting
- LocalStorage persistence
- Undo delete
- Filtering & search

## Tech Stack
- React (Vite)
- Tailwind CSS
- LocalStorage
- useReducer + custom hook
